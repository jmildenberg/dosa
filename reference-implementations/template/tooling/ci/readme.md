# Template :: Tooling :: CI

> [Template](../../readme.md) :: [Tooling](../readme.md) :: CI

Pipeline definitions that run on every push to main under
[Trunk-Based Development](../../../../docs/adrs/0006-adopt-trunk-based-development.md): the
[Testing Trophy](../../../../docs/adrs/0007-testing-trophy-and-test-classification.md) suites, the
security gates from [Security](../../../../docs/security.md), the image build in
[deploy/docker/](../../deploy/docker/readme.md), and the promotion into the first environment of the
[delivery progression](../../../../docs/adrs/0008-standard-delivery-progression.md).
