# Template :: Documentation :: Operations

> [Template](../../readme.md) :: [Documentation](../readme.md) :: Operations

This domain's runbooks and operational specifics, satisfying the org-wide
[Operations](../../../../docs/operations.md) model: one runbook per alertable condition (what the alert
means, how to verify it, and the remediation), the SLOs this domain commits to, and its disaster-recovery
restore procedure. Runbooks are versioned and reviewed like code — a stale runbook is a defect, not a
documentation gap to get to later.
