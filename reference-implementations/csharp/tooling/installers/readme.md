# C# :: Tooling :: Installers

> [C#](../../readme.md) :: [Tooling](../readme.md) :: Installers

Scripts that set up a local development environment: installing the toolchain version this domain
builds with, wiring up [githooks/](../githooks/readme.md), and standing up any local dependencies (e.g.
a local database container matching [databases/](../../databases/readme.md)) needed to run the domain
outside CI.
